# 💻 Abhishek Jain — Portfolio Website

Welcome to my personal portfolio website built with **React.js**, **Framer Motion**, and modern UI principles.  
It showcases my projects, skills, and journey as a **MERN stack developer** expanding into **DevOps and backend systems engineering**, with a long-term goal of working on AI-powered products.


## 🚀 Features

* 🎨 Clean, modern UI with dark theme
* ⚡ Smooth Framer Motion animations
* 💼 Projects, Resume, and Contact sections
* 🧠 Tech-focused portfolio highlighting full-stack & backend work
* 📨 Functional contact form using **EmailJS**
* 🧩 Fully responsive for all screen sizes
* 🚀 Deployment-ready setup
